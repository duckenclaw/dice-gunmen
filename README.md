# dice-gunmen

The game starts on a 2D plane with 2 player characters. The game is played in turns. First turn is decided by a d6 roll contest.

Turn:
1. Roll D6, receive **AP** *(Action Points)* equal to the number on the dice
2. Use actions
3. End turn

**Actions:**
- **Shoot.** Enter aiming stance, the movement controls now dictate which direction the character is aiming at. Press "SPACE" to fire a projectile in the chosen direction that will destroy the first obstacle in the way or kill a player.
- **Move.** Move to an adjacent grid not occupied by any obstacles or players.\

Rules:
- The last player alive wins
